consolidate(type, symbol, size, tick_type, handler)[source]

Registers the handler to receive consolidated data for the specified symbol

Parameters:
  • type (PyObject) — The Python type of the bar (RenkoBar, VolumeRenkoBar, or RangeBar)
  • symbol (Symbol) — The symbol whose data is to be consolidated
  • size (float) — The size value for the consolidator (e.g., brick size, range size or maxCount)
  • tick_type (TickType) — The tick type to consolidate. If null, the first matching subscription is used.
  • handler (PyObject | None) — The callback to invoke with each new consolidated bar
Returns:

The created and registered IDataConsolidator instance
