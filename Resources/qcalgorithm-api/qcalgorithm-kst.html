kst(symbol, roc_1_period=10, roc_1_ma_period=10, roc_2_period=15, roc_2_ma_period=10, roc_3_period=20, roc_3_ma_period=10, roc_4_period=30, roc_4_ma_period=15, signal_period=9, moving_average_type=0, resolution=None, selector=None)[source]

Creates a new KnowSureThing indicator for the symbol. The indicator will be automatically updated on the given resolution.

Parameters:
  • symbol (Symbol) — The symbol whose KST we want
  • roc_1_period (int, optional) — The period over which to compute ROC1
  • roc_1_ma_period (int, optional) — The smoothing period used to smooth the computed ROC1 values
  • roc_2_period (int, optional) — The period over which to compute ROC2
  • roc_2_ma_period (int, optional) — The smoothing period used to smooth the computed ROC2 values
  • roc_3_period (int, optional) — The period over which to compute ROC3
  • roc_3_ma_period (int, optional) — The smoothing period used to smooth the computed ROC3 values
  • roc_4_period (int, optional) — The period over which to compute ROC4
  • roc_4_ma_period (int, optional) — The smoothing period used to smooth the computed ROC4 values
  • signal_period (int, optional) — The smoothing period used to smooth the signal values
  • moving_average_type (MovingAverageType, optional) — Specifies the type of moving average to be used as smoother for KnowSureThing values
  • resolution (Resolution, optional) — The resolution
  • selector (Callable[IBaseData, float], optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
Returns:

A new KnowSureThing indicator with the specified smoothing type and period

Return type:

KnowSureThing
