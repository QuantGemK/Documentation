kvo(symbol, fast_period, slow_period, signal_period=13, resolution=None, selector=None)[source]

Creates a new Klinger Volume Oscillator (KVO) indicator

Parameters:
  • symbol (Symbol) — The symbol whose KVO we want
  • fast_period (int) — The period of the fast EMA used to calculate KVO
  • slow_period (int) — The period of the slow EMA used to calculate KVO, default to 13
  • signal_period (int, optional) — The period of the signal EMA of the raw KVO value
  • resolution (Resolution, optional) — The resolution.
  • selector (Callable[IBaseData, TradeBar], optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
Returns:

The Klinger Volume Oscillator indicator for the requested symbol.

Return type:

KlingerVolumeOscillator
