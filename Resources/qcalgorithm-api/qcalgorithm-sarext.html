Creates a new Parabolic SAR Extended indicator

Parameters:
  • symbol (Symbol) — The symbol whose SAREXT we seek
  • sar_start (float, optional) — The starting value for the Stop and Reverse indicator
  • offset_on_reverse (float, optional) — The offset value to be applied on reverse
  • af_start_short (float, optional) — The starting acceleration factor for short positions
  • af_increment_short (float, optional) — The increment value for the acceleration factor for short positions
  • af_max_short (float, optional) — The maximum value for the acceleration factor for short positions
  • af_start_long (float, optional) — The starting acceleration factor for long positions
  • af_increment_long (float, optional) — The increment value for the acceleration factor for long positions
  • af_max_long (float, optional) — The maximum value for the acceleration factor for long positions
  • resolution (Resolution, optional) — The resolution
  • selector (Callable[IBaseData, IBaseDataBar], optional) — Selects a value from the BaseData to send into the indicator, if null defaults to casting the input value to a TradeBar
Returns:

A ParabolicStopAndReverseExtended configured with the specified periods

Return type:
